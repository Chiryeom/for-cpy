# ⚖️ 左然 互动展示网页

这是一个为《未定事件簿》角色——左然，量身定制的交互式网页。采用轻快柔和的浅粉色调，搭配精致的 CSS 像素装饰图标，旨在提供一个温馨、私密的互动空间。

## ✨ 核心功能

- **🌸 个性化登录**：进入网页时可输入专属昵称，开启个性化互动体验。
- **🎴 趣味互动选择**：
  - **心动翻牌**：通过翻牌子的形式随机选定互动对象（目前专注于左然）。
  - **朕都要**：直接查看角色卡面列表进行选择。
- **🖼️ 互动抽取系统**：
  - **心动瞬间**：随机抽取左然的精美思绪/邀请函卡面。
  - **我想对你说**：随机抽取左然的深情台词（收录了多条电影感十足的经典语录）。
  - **图鉴库**：一键查看所有已收录的精美卡面。
- **🔮 今日运势**：带有精致加载动画的随机运势抽取功能，每日为你送上一份小惊喜。
- **💎 视觉美学**：
  - 采用细体字与浅粉色背景，营造高级感。
  - 纯 CSS 绘制的像素图标（骰子、可乐、天平、钻石），细节满满。

## 🚀 快速开始

### 1. 配置图片素材
由于版权原因，项目未内置高清原图。请将您的左然卡面图片放入 `assets/images/` 文件夹。
建议按照 `js/characters.js` 中的路径进行命名，例如：
- `autumn.jpg` (秋叶如织)
- `breath.jpg` (此时于此刻)
- ...

### 2. 自定义台词
您可以随时打开 `js/characters.js`，在 `quotes` 数组中添加更多您喜欢的左然语录。

### 3. 本地预览
直接使用浏览器打开 `index.html` 即可开始你的浪漫之旅。

## 🎨 界面定制

项目在 `css/style.css` 中定义了全局变量，您可以轻松修改主题色：
```css
:root {
    --primary-pink: #f8c2d1;      /* 主色调 */
    --accent-pink: #ff85a2;       /* 强调色 */
    --bg-light: #fff5f7;          /* 背景色 */
}
```
